## Overview

Houdini Professional Suite is a complete VFX production platform with pipeline integration, render management, and collaboration tools for studio-scale production.

Houdini Engine integrates Houdini's procedural tools into other applications including Unity, Unreal Engine, and Maya, enabling procedural content generation within game engines and other DCC tools. Digital assets package procedural networks as reusable tools that non-Houdini users can operate.

Solaris provides a USD-based scene building and rendering environment that handles the lighting, look development, and final rendering stage of the VFX pipeline with support for multiple render engines including Karma, RenderMan, and Arnold.

PDG (Procedural Dependency Graph) automates and distributes computation-heavy tasks across render farms and cloud resources, managing the processing pipeline that large-scale VFX production requires.

Houdini Professional Suite is the VFX platform for studios that need the most powerful procedural tools in the industry with the pipeline integration and distributed processing that feature-film-scale production demands.

## Performance & Integration

Houdini is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Houdini processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
